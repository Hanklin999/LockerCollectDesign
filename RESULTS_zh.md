## 1. 這個研究在做什麼？

想像你是一間電商公司，你在全台灣放了 2,000 台智慧取件機器。

買家下單之後，包裹會被放進機器裡。機器會發簡訊通知買家：「你的包裹到了，請來取。」

問題來了：

> **發越多次簡訊，買家就會越快來取嗎？**

這個問題聽起來很簡單，但其實不容易回答。

因為：**快速取件的買家，可能本來就是習慣好的人，不是因為你多發了簡訊。**

這個研究用「因果推斷」的方法，把這兩件事分開，找出「多發簡訊」本身真正的效果。

---

## 2. 我們的問題是什麼？

### 表面問題
發 2 次簡訊 vs 發 3 次 vs 發 5 次，哪一種讓買家最快來取件？

### 真正的問題（因果問題）
**排除掉門市本身的差異之後**，簡訊通知次數對取件速度的「因果效應」是多少？

### 為什麼要這樣問？

舉個例子：

| 門市 | 通知次數 | 平均取件時間 |
|------|---------|------------|
| 台北信義門市 | 5 次 | 30 小時 |
| 宜蘭冬山門市 | 2 次 | 38 小時 |

看起來「發 5 次」讓大家更快取件。但問題是：台北門市的買家本來就比較忙碌、習慣快速取件。就算只發 2 次，台北門市的取件速度也可能比宜蘭快。

所以我們需要更嚴謹的方法，把「門市本身的差異」從結果裡扣掉，才能知道「多發簡訊」真正的貢獻。

## 3. 實驗設計

### 3.1 實驗對象

全台灣 2,000 間智慧取件門市，分成兩大類：

| 類型 | 說明 | 數量 |
|------|------|------|
| 爆店（burst） | 庫存量高，常常快滿的門市 | 1,000 間 |
| 空店（vacant） | 庫存量低，很少爆滿的門市 | 1,000 間 |

### 3.2 怎麼分組？

**爆店（1,000 間）** 隨機分成 4 組：

```
┌──────────────────────────────────────────────────────────┐
│  5D 對照組（Control）  │ 5 天截止 │ 發 2 次 │ 100 間   │
│  5D G2               │ 5 天截止 │ 發 3 次 │ 100 間   │
│  5D G4               │ 5 天截止 │ 發 5 次 │ 100 間   │
│  6D                  │ 6 天截止 │ 發 2 次 │ 700 間   │
└──────────────────────────────────────────────────────────┘
```

**空店（1,000 間）** 全部分到一組：

```
┌──────────────────────────────────────────────────────────┐
│  7D                  │ 7 天截止 │ 發 2 次 │ 1,000 間  │
└──────────────────────────────────────────────────────────┘
```

### 3.3 通知時間點說明

| 組別 | 第幾天發簡訊 | 總共幾封 |
|------|------------|---------|
| Control | D0（到貨當天）、D4（截止前一天）| 2 封 |
| G2 | D0、D2、D4 | 3 封 |
| G4 | D0、D1、D2、D3、D4（每天都發）| 5 封 |

> **D0** = 包裹到達機器的那天
> **D4** = 截止日前一天（5 天截止的話，D4 就是第 4 天）

### 3.4 為什麼 G4 叫 G4 不叫 G5？

原本有規劃 G3（發 4 封，D0、D1、D2、D4），但因為成本考量和分析複雜性，最終被捨棄。所以只剩 G2 和 G4。

### 3.5 實驗時間

| 日期 | 角色 | 說明 |
|------|------|------|
| 2026-01-19 | 基準週（BM） | 實驗還沒開始，用來比較用的基準 |
| 2026-01-26 | 實驗第 1 週 | 開始發不同頻率的通知 |
| 2026-02-02 | 實驗第 2 週 | 繼續 |
| 2026-02-09 | **排除** | 過年前一週，買家特別勤快取件，是外部因素 |
| 2026-02-23 | 實驗第 3 週 | 繼續 |
| 2026-03-02 | 實驗第 4 週 | 實驗結束 |

## 5. 研究方法

我們用了四種方法，從不同角度回答同一個問題。就像法官審案，要從多個證人那裡得到一致的說詞，才能確認結論。

### 5.1 方法一：差異中的差異（DiD）

**用一句話說**：把「時間趨勢」扣掉，看剩下的差異是不是通知造成的。

**類比**：
- 假設全台灣的取件速度在實驗期間都會因為氣候或習慣而慢慢變快
- 我們要排除這個「自然趨勢」，只看通知策略本身的效果

**公式長這樣（不用背，看意思就好）**：
```
取件時間 = 門市差異 + 時間差異 + 通知效果 + 誤差
```

**關鍵假設**：如果沒有改變通知策略，對照組和實驗組的取件速度應該會以同樣的速度變化（這叫「平行趨勢假設」）。

**我們驗證了這個假設**：基準週（實驗前）各組的數字很接近，符合要求。

---

### 5.2 方法二：傾向分數匹配（PSM）

**用一句話說**：找「最像」的對照組門市來比較，讓比較更公平。

**問題在哪**：實驗組和對照組的門市本來就不一樣（庫存量、流量等），直接比不公平。

**PSM 怎麼做**：
1. 計算每間門市被分到實驗組的「機率」（用門市特性預測）
2. 幫每間實驗組門市，找一間機率最接近的對照組門市配對
3. 配對完的兩組門市特性很接近了，再比較結果

**類比**：
- 你要比較「吃減肥藥」vs「沒吃」的效果
- 但吃藥的人本來就比較重
- PSM 幫你找「體重一樣重但沒吃藥」的人來比較，這樣才公平

---

### 5.3 方法三：異質性處理效應（HTE）

**用一句話說**：平均效果是 -1.5 小時，但不是每間門市都一樣，哪些門市效果最大？

**為什麼重要**：如果預算有限，我們應該先把資源給「效果最好」的門市。

**用了四種機器學習方法**：

| 方法 | 優點 | 缺點 |
|------|------|------|
| T-Learner | 最直觀 | 小樣本時不穩定 |
| S-Learner | 用所有資料 | 容易低估效果 |
| X-Learner | 最適合本實驗 | 計算較複雜 |
| Causal Forest DML | 能給出信賴區間 | 最耗時 |

---

### 5.4 方法四：敏感度分析

**用一句話說**：用各種方法「試圖推翻」我們的結論，看它撐不撐得住。

**四個測試**：

| 測試 | 問的問題 |
|------|---------|
| Rosenbaum Bounds | 如果有我們沒觀察到的因素，要多強才能推翻結論？|
| 安慰劑測試 | 通知策略有沒有影響到它「不應該」影響的指標？|
| 留一週出去（LOO）| 結論是不是靠某一週特別的數字撐起來的？|
| 係數穩定性 | 加入更多控制變數，結論會不會改變？|

## 6. 實驗數據

### 6.1 基本數字

**實驗後（4 週平均）各組取件時間**

| 組別 | 平均取件時間 | 比對照組快多少 |
|------|------------|--------------|
| 5D Control（2 封，對照）| 33.72 小時 | — |
| 5D G2（3 封）| 32.04 小時 | **快 1.68 小時** |
| 5D G4（5 封）| 31.91 小時 | **快 1.81 小時** |
| 6D（6 天截止）| 33.99 小時 | 比對照慢 0.27 小時（正常，截止較寬）|
| 7D（空店）| 37.51 小時 | 空店基準本來就高 |

**退件率（RTS）**

| 組別 | 退件率 |
|------|-------|
| 5D Control | 1.716% |
| 5D G2 | 1.305% |
| 5D G4 | 1.404% |

---

### 6.2 DiD 結果

**主要 DiD 估計（Two-Way Fixed Effects）**

| 比較 | 效果（小時）| 標準誤差 | P 值 |
|------|-----------|---------|------|
| G2 + G4 vs Control | **-1.505 小時** | 0.238 | < 0.001 |
| G2 vs Control | **-1.228 小時** | 0.277 | < 0.001 |
| G4 vs Control | **-1.782 小時** | 0.262 | < 0.001 |
| G4 比 G2 多縮短多少（邊際效益）| -0.555 小時 | — | < 0.001 |

> **P 值 < 0.001** 的意思：這個效果不是巧合的機率不到 0.1%，非常可信。

**每週的效果（Event Study）**

| 週次 | 效果（小時）| 解讀 |
|------|-----------|------|
| 基準週（第 0 週）| 0（基準）| 實驗還沒開始，當然是 0 |
| 第 1 週 | -1.390 | 通知策略立刻就有效果 |
| 第 2 週 | -1.395 | 效果持續 |
| 第 3 週 | -1.431 | 效果略微增強 |
| 第 4 週 | -1.803 | 效果最強（可能是習慣累積）|

**排列測試（確認不是巧合）**

把分組隨機打亂 500 次，看看「亂猜」能不能猜出 -1.5 小時的效果：

```
真實效果：-1.505 小時
500 次亂猜的平均：+0.018 小時（幾乎是 0）
P 值：< 0.001（500 次裡沒有一次亂猜出接近 -1.5 的數字）
```

這確認了我們的結果不是運氣好猜到的。

### 6.5 敏感度分析結果

**Rosenbaum Bounds（隱藏混淆因子的容忍度）**

| Γ 值 | 最壞情況 P 值 | 結論還成立？|
|------|------------|-----------|
| 1.0（無隱藏偏差）| < 0.001 | ✅ |
| 1.5 | < 0.001 | ✅ |
| 2.0 | < 0.001 | ✅ |
| 2.5 | < 0.001 | ✅ |
| 3.0 | < 0.001 | ✅ |

> 臨界 Γ 超過 3.0：隱藏因素要讓實驗組被選到的機率超過對照組 **3 倍**，才有可能推翻我們的結論。這是非常強的穩健性。

**安慰劑測試**

| 指標 | 效果 | 解讀 |
|------|------|------|
| 取件時間（真實指標）| -1.505 小時 ✅ | 正確，這就是我們要看到的 |
| 投訴率（安慰劑）| +0.001 個百分點 | 數字很小，見說明 |
| 退訂率（安慰劑）| +0.000 個百分點 | 數字很小，見說明 |

> **為什麼安慰劑有效果？** 因為 G2/G4 組本來就多發通知，所以投訴和退訂率在實驗前就略高。這是通知頻率的「直接副作用」，不是 confounding（混淆）的問題。效果極小（< 0.001%），不影響主要結論。這反而提醒我們要監控這兩個護欄指標。

**留一週出去（LOO）測試**

| 排除哪週 | 估計效果 | 與完整樣本差距 |
|---------|---------|-------------|
| 完整樣本 | -1.505 小時 | — |
| 排除第 1 週 | -1.540 小時 | -0.035 小時 |
| 排除第 2 週 | -1.540 小時 | -0.035 小時 |
| 排除第 3 週 | -1.529 小時 | -0.024 小時 |
| 排除第 4 週 | -1.397 小時 | +0.108 小時 |

> 最大變動只有 0.108 小時，結論不依賴任何單一週的數字。

**係數穩定性**

| 加入的控制變數 | 估計效果 |
|-------------|---------|
| 無控制變數 | -1.505 小時 |
| + 時間固定效應 | -1.505 小時 |
| + 門市固定效應 | -1.505 小時 |
| + 所有共變數 | -1.505 小時 |

> 不管加不加控制變數，效果完全一樣，代表沒有遺漏重要的混淆因素。

## 7. 解讀數據

### 7.1 主要發現是什麼？

**發 3 封通知（G2）比發 2 封（對照組）有效：平均快 1.5 小時。**

這個效果：
- 統計上高度顯著（P < 0.001）
- 在兩個不同方法（DiD 和 PSM）下都出現
- 能承受嚴格的穩健性測試

### 7.2 發更多封會更好嗎？

**不一定——邊際效益遞減了。**

```
2 封 → 3 封：快 1.2 小時（很划算）
3 封 → 5 封：再快 0.5 小時（效益很小）
```

換句話說，**第 4、5 封通知的貢獻非常有限**，但成本和買家的反感卻是真實存在的。

### 7.3 哪些門市效果最大？

**反直覺的發現：非六都門市的效果比六都更大。**

```
非六都門市效果：-2.4 小時
六都門市效果：  -1.7 小時
```

為什麼？可能的解釋：非六都的買家原本取件習慣較差，被通知提醒後改善幅度更大。六都的買家本來就比較忙碌、習慣好，改善空間較小。

**最重要的特性是門市容量，不是地點。**

門市能裝越多包裹，通知策略的效果越明顯。地點（六都 vs 非六都）幾乎不重要。

### 7.4 護欄指標要注意嗎？

**要，但不用擔心。**

發越多封通知，投訴率和退訂率確實會略微上升，但效果極小（不到 0.001 個百分點），在 800 萬/天的規模下是可以接受的，但仍需持續監控。

## 8. 結果推論

### 8.1 業務建議

**建議一：全面推行 G2（3 封通知）**

理由：
- 每個包裹快取件 1.5 小時
- 95% 的門市（285/300）都有明顯效益
- 退件率降低 0.36 個百分點 = 每天少退 2,880 件
- 相較 G4，成本低、副作用小

**建議二：不建議全面推行 G4（5 封通知）**

理由：
- G4 比 G2 只多縮短 0.5 小時
- 但要多發 2 封通知，成本增加
- 投訴率和退訂率略微上升
- 投資報酬率不如 G2

**建議三：如果資源有限，優先升級非六都門市**

理由：
- 非六都效果（-2.4 小時）遠大於六都（-1.7 小時）
- 同樣的資源投入，非六都的回報更高

### 8.2 整體評分

| 測試項目 | 結果 | 評分 |
|---------|------|------|
| DiD 估計顯著 | -1.505 小時，p < 0.001 | ✅ 通過 |
| PSM 與 DiD 一致 | 差距只有 9% | ✅ 通過 |
| 排列測試 | p < 0.001 | ✅ 通過 |
| Rosenbaum Γ | > 3.0（非常強）| ✅ 通過 |
| LOO 穩定性 | 最大變動 0.1 小時 | ✅ 通過 |
| 係數穩定性 | 0% 變化 | ✅ 通過 |
| 安慰劑測試 | 有解釋，影響極小 | ⚠️ 可接受 |

**結論**：在 6 項測試中 6 項通過，1 項有合理解釋，研究結論是**可信的**。

## 9. 研究限制

### 9.1 只有一個基準週

我們只有一週的「實驗前資料」，沒辦法正式驗證「平行趨勢假設」（DiD 的核心假設）。

**怎麼處理**：我們檢查了基準週各組的門市特性，確認差異很小，並用 event study 顯示效果從實驗開始就出現（不是基準週就有）。

### 9.2 使用模擬資料

真實公司資料因為保密問題無法使用，所以用模擬資料。

**怎麼處理**：所有參數都是根據真實觀察值校準的，DGP 完全透明、可重現。

### 9.3 每組樣本數較小

每組只有 100 間門市，隨機分配後仍有殘餘的不平衡（G4 和對照組的幾個特性差距略大）。

**怎麼處理**：PSM 把這些不平衡修正到 SMD < 0.1，兩個方法得到一致結論。

### 9.4 排除了過年週

過年前一週（2026-02-09）的取件速度特別快，這是季節性因素，不是通知效果。

**怎麼處理**：合理排除，且 LOO 測試確認沒有任何單一週在驅動結果。

## 10. 名詞解釋

| 名詞 | 用國中生能懂的話說 |
|------|-----------------|
| **因果推斷** | 不只看「A 和 B 一起出現」，而是確認「A 真的造成了 B」|
| **DiD（差異中的差異）** | 把時間趨勢扣掉，看剩下的差異是不是我們的介入造成的 |
| **PSM（傾向分數匹配）** | 幫每個實驗組門市，找一個「最像它但沒有被介入」的對照，再比較 |
| **HTE（異質性效應）** | 平均效果之外，找出「誰受益最多、誰受益最少」|
| **Rosenbaum Bounds** | 測試「就算有我們不知道的因素，結論會不會被推翻」|
| **平行趨勢假設** | 假設「如果沒有介入，實驗組和對照組會以一樣的速度改變」|
| **固定效應（Fixed Effects）** | 把每個門市、每個時間點的固有差異都控制掉 |
| **SMD** | 衡量兩組人的某個特性差多少的標準化數字，< 0.1 代表很平衡 |
| **P 值** | 「這個結果是純屬巧合」的機率，< 0.05 代表很可信 |
| **信賴區間（CI）** | 真實效果「最可能落在哪個範圍」的估計 |
| **ATE** | 對所有人的平均效果 |
| **ATT** | 對「有收到介入」的人的平均效果 |
| **CATE** | 對每一個個別單位的個別效果估計 |
| **RTS 退件率** | 超過截止時間、被退回給賣家的包裹比例，越低越好 |
| **護欄指標** | 確保介入沒有帶來副作用的監控指標 |
| **LOO 測試** | 每次拿掉一週的資料重新估計，看結論會不會大幅改變 |
| **安慰劑測試** | 把效果放在「不應該有效果」的指標上，確認我們的結論不是假象 |
| **DGP** | Data Generating Process，資料是怎麼被產生出來的完整說明 |
| **基準週（BM）** | 實驗開始前的那週，用來當作比較基準 |
| **Γ（Gamma）** | Rosenbaum Bounds 的關鍵數字，越大代表結論越穩健 |
